# Iocn-Library

### npm install --registry=https://registry.npm.taobao.org

### 需要MongoDB

### 在config/config.js下配置服务器与数据库相关信息

### npm run dev

### 另开终端，npm run server

### 打开localhost:3000，（默认），初始化groups

### 然后修改src/modules/Views/Index.js，把初始化groups的代码去掉

### npm run build

### localhost:3000，（默认）, Enjoy Yourself !
